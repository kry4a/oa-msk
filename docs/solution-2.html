Решения для компании RusGems

Один из крупнейших поставщиков продукции для ювелирных и ограночных производств

Заказчик:

Группа компаний «Rusgems» является официальным представителем в России компании «Dalloz» - ведущего мирового производителя машинной огранки фианитов, корундов и синтетической шпинели. Вставки отличаются высокоточной калибровкой, отличной полировкой граней и могут быть использованы при литье изделий. Одним из крупнейших поставщиков на российский рынок продукции для ювелирных и ограночных производств.

Отрасль: производство ювелирных изделий, оптовая торговля

Внедренные решения

Разработка по индивидуальному заказу:

  • Управление складом
  • Управление отношениями с клиентами (CRM)
  • Управление закупками (снабжение) и отношениями с поставщиками.
  • Взаиморасчеты с поставщиками
  • Учет прихода ТМЦ
  • Различная отраслевая специфика
  • Производство, услуги
  • Расчет фактической себестоимости
  • Учет затрат на производство (РАУЗ)
  • Учет услуг производственного характера
  • Управление продажами, логистикой и транспортом (SFM, WMS, TMS)
  • Продажи (сбыт), сервис, маркетинг
  • Взаиморасчеты с покупателями
  • Оптовая торговля
